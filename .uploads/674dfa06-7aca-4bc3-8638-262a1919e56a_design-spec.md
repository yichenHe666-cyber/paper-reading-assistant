#### CDN 来源

```
Google Fonts: Kalam, Caveat, Ma Shan Zheng, JetBrains Mono, Quicksand
jsDelivr: lxgw-wenkai-webfont@1.7.0/style.css
KaTeX: katex@0.16.11 (CSS + JS + auto-render)
```

#### 字体渲染优化（全局）

```css
html {
  -webkit-font-smoothing: antialiased;
  -moz-osx-font-smoothing: grayscale;
  text-rendering: optimizeLegibility;
  font-smooth: never;
}

body {
  text-shadow: 0 0 0.5px rgba(0, 0, 0, 0.15);
}
```

#### 渐变文字清晰度方案

- 不使用 `filter: drop-shadow()`（会导致文字模糊）
- 使用 `-webkit-text-stroke: 0.2~0.3px rgba(color, 0.06~0.08)` + `paint-order: stroke fill`

## 2. 全局效果

### 2.1 "五彩斑斓的黑" 文字彩色叠层

全屏 `body::before` 伪元素，极低透明度多彩渐变 + `mix-blend-mode: multiply`，让深色文字呈现微妙的彩色光泽。

```css
body::before {
  content: '';
  position: fixed;
  inset: 0;
  pointer-events: none;
  z-index: 9999;
  background: linear-gradient(
    120deg,
    rgba(255, 110, 180, 0.03) 0%,
    rgba(168, 85, 247, 0.035) 15%,
    rgba(78, 205, 196, 0.03) 30%,
    rgba(236, 72, 153, 0.025) 45%,
    rgba(110, 181, 255, 0.04) 60%,
    rgba(192, 132, 252, 0.03) 75%,
    rgba(255, 107, 107, 0.025) 90%,
    rgba(255, 110, 180, 0.03) 100%
  );
  background-size: 400% 100%;
  animation: rainbowBlackFlow 12s linear infinite;
  mix-blend-mode: multiply;
}

@keyframes rainbowBlackFlow {
  0% { background-position: 0% 50%; }
  100% { background-position: -200% 50%; }
}

/* 修复叠层挡住交互 */
body > * { position: relative; z-index: 1; }
body > .particles-container { z-index: 0; }
```

### 2.3 浮动粒子（纯 CSS，20 个）

- 容器 `.particles-container`: `position: fixed; inset: 0; pointer-events: none; z-index: 0`
- 每个粒子 `.particle`: `position: absolute; border-radius: 50%`
- 颜色交替: `neon-cyan` / `neon-magenta` / `neon-yellow` / `neon-purple`
- 尺寸: `3~5px`
- 动画: `particleFloat` 线性循环，时长 `15~26s`，随机延迟

```css
@keyframes particleFloat {
  0%   { transform: translateY(110vh) translateX(0) scale(0); opacity: 0; }
  10%  { opacity: 0.8; transform: translateY(90vh) translateX(20px) scale(1); }
  50%  { opacity: 0.5; transform: translateY(50vh) translateX(-15px) scale(0.8); }
  90%  { opacity: 0.3; transform: translateY(10vh) translateX(10px) scale(0.6); }
  100% { transform: translateY(-10vh) translateX(-5px) scale(0); opacity: 0; }
}
```

## 3. 核心动画清单

| 动画名 | 用途 | 时长 | 缓动 | 关键帧 |
|--------|------|------|------|--------|
| `neonPulse` | 霓虹呼吸脉冲（广泛使用） | 2~3s | ease-in-out, infinite | `brightness(1) -> brightness(1.3)` |
| `biliDanmakuFlow` | B站弹幕流光渐变文字 | 8~12s | linear, infinite | `background-position: 100% -> -100%` |
| `holoShimmer` | 全息卡片微光扫过 | 8s | ease, infinite | `background-position: 200% -> -200%` |
| `floatUp` | 悬浮上下浮动 | 3s | ease-in-out, infinite | `translateY(0) -> translateY(-8px)` |
| `sparkle` | 闪烁装饰 | -- | ease, infinite | `scale(0) rotate(0) -> scale(1) rotate(180deg)` |
| `twinkle` | 眨眼星光 | 1.4~2s | ease-in-out, infinite | `opacity: 0.2, scale: 0.5 -> opacity: 1, scale: 1.2` |
| `spin` | 旋转（同步图标） | 1s | linear | `rotate(0) -> rotate(360deg)` |
| `borderGlow` | 边框颜色循环 | 4s | ease, infinite | `cyan -> purple -> magenta` |
| `fadeInUp` | 消息气泡入场 | 0.3s | ease | `opacity: 0, translateY(20px) -> opacity: 1, translateY(0)` |
| `slideInLeft` | 左侧滑入 | -- | ease | `opacity: 0, translateX(-20px) -> opacity: 1, translateX(0)` |
| `slideInRight` | 右侧滑入 | -- | ease | `opacity: 0, translateX(20px) -> opacity: 1, translateX(0)` |
| `rainbowBlackFlow` | 五彩斑斓的黑 | 12s | linear, infinite | `background-position: 0% -> -200%` |
| `particleFloat` | 粒子上浮 | 15~26s | linear, infinite | 见 [2.3 浮动粒子](#23-浮动粒子纯-css20-个) |
| `gradientShift` | 渐变位移（备用） | -- | ease, infinite | `0% -> 50% -> 100%` |
| `streamingDots` | 流式打字指示 | -- | -- | `content: '' -> '.' -> '..' -> '...'` |
| `dmScrollLeft` | 弹幕滚动 | 6~10s | linear, infinite | `translateX(0) -> translateX(calc(-100% - 100vw))` |

## 6. 关键实现注意事项

### 6.1 避免渐变文字模糊

- **禁止** 在渐变文字（`background-clip: text`）上使用 `filter: drop-shadow()`
- 使用 `-webkit-text-stroke` + `paint-order: stroke fill` 替代
- 值范围: stroke `0.2~0.3px`, alpha `0.06~0.08`

### 6.2 手写体清晰度

- 全局 `font-smooth: never`（手写体不需要平滑）
- body 加 `text-shadow: 0 0 0.5px rgba(0, 0, 0, 0.15)`
- 灰色文字（secondary/muted）颜色不能太浅，建议 secondary 最低 `#3a3a5a`
- 论文概述等场景额外加 `text-shadow: 0 0 0.3px rgba(0, 0, 0, 0.2)`

### 6.3 五彩斑斓的黑

- `body::before` z-index: `9999`, `pointer-events: none`
- `body > *` 需要 z-index: `1` 确保可交互
- `.particles-container` 需要 z-index: `0`

### 6.4 KaTeX 注意事项

- `ignoredTags` 中 **不含** `code`，确保行内公式可以在 code 元素内渲染
- 含 `pre`，代码块内容不会被 KaTeX 扫描

### 6.5 Brand Icon 注意事项

- 在 `.nav-brand` 内使用时需覆盖品牌渐变文字的透明填充:

```css
.nav-brand .brand-icon {
  -webkit-text-fill-color: initial;
}
```
